## Git командалары

Бұл жобада келесі Git командалары қолданылды:

1. git clone  
Репозиторийді GitHub-тан компьютерге көшіру үшін.

2. git add .  
Өзгертілген файлдарды commit-ке дайындау үшін.

3. git commit -m "message"  
Өзгерістерді сақтау (commit жасау).

4. git push  
Commit-терді GitHub репозиторийіне жіберу үшін.

5. git pull  
Репозиторийден соңғы өзгерістерді алу үшін.
